## 运行&打包

``` bash
# install dependencies 下载依赖
npm install

# serve with hot reload at localhost:8080 运行开发环境
npm run dev

# build for production with minification 打包
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
